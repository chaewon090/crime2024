# 서울 범죄 분석 대시보드 
## 1. 사용한 프롬프트 공유 링크 - (https://ai.studio/apps/3c186b8a-0393-487e-810c-a25e1c52e1d4)

차트1) 사용 sql:

SELECT c.자치구, c.CCTV수, f.절도 
FROM CCTV현황 c 
JOIN 경찰서관할 p 
    ON c.자치구 = p.자치구 
JOIN 범죄현황 f 
    ON p.경찰서 = f.경찰서 
WHERE c.CCTV수 IS NOT NULL 
      AND f.절도 IS NOT NULL;
데이터 처리:
2024년 경찰서 기준 범죄 데이터를 경찰서 관할 자치구 기준으로 연결하였다.
단, 경찰서와 자치구가 명확히 1:1 대응되지 않는 경우는 제외하였다.


차트 2) 사용 sql:

SELECT
    p.경찰서,
    (p."2024년 지구대" + p."2024년 파출소" + p."2024년 치안센터") AS 관서수,
    SUM(c.건수) AS 전체범죄발생건수
FROM police_station p
JOIN crime_2024 c
    ON p.경찰서 = c.구분
WHERE c.발생검거 = '발생'
GROUP BY
    p.경찰서,
    p."2024년 지구대",
    p."2024년 파출소",
    p."2024년 치안센터";

차트 3) 사용 sql:

SELECT
    범죄유형,
    SUM(건수) AS 발생건수
FROM crime_2024
WHERE 발생검거 = '발생'
GROUP BY 범죄유형
ORDER BY 발생건수 DESC;






## 2. 데이터 및 전처리 과정 ### 사용 데이터 - 서울 경찰서별 5대 범죄 발생 및 검거 데이터 (2024년) - 서울시 자치구 CCTV 설치 현황 - 경찰서별 지역경찰 관서 현황 ### 전처리 과정 - 자치구와 경찰서 단위 불일치를 해결하기 위해 매핑 테이블 생성 - CCTV 데이터를 경찰서 기준으로 변환 (SUM) - 관서 데이터를 합산하여 총 관서 수 생성 - 범죄 데이터를 발생 기준으로 재구성 - 2024년 데이터만 추출하여 분석 --- 
## 3. 시각화 결과 및 인사이트 ### (1) CCTV 수와 절도 발생량 - CCTV 수와 절도 발생 간 뚜렷한 감소 관계는 나타나지 않음 ### (2) 관서 수와 범죄 발생량 - 관서 수가 많은 지역에서 범죄도 많은 경향 존재 ### (3) 범죄 유형별 발생 비율 - 절도와 폭력 범죄 비중이 높게 나타남 --- ## 4. 결론 - CCTV 및 경찰 인프라와 범죄 간 단순 인과관계는 확인 어려움 - 지역 특성을 고려한 추가 분석 필요
